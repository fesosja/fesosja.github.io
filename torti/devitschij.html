Рецепты

Назад

Торт "Девичий"

Ингредиенты

Для коржей:

  • три яйца
  • один стакан сахара
  • половина банки сгущёнки
  • одна банка майонеза
  • одна чайная ложка погашенной соды
  • мука

Для крема:

  • стакан сметаны
  • половина стакана сахара
  • сок лимона
  • измельченные орехи

Приготовление

Яйца взбить с сахаром, добавить сгущёнку, майонез, соду. Добавить муку, стобы тесто получилось густоты сметаны. Испечь три коржа, в один из них добавить предварительно две ложки какао.
