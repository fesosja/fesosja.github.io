Рецепты

Назад

Торт "Мишка на севере"

Ингредиенты

Для коржей:

  • 300 г сметаны
  • 1/5 стакана сахара
  • половина чайной ложки соды
  • одна столовая ложка уксуса
  • три стакана муки

Для крема:

  • один стакан сметаны
  • 1/5 стакана сахара

    • Для глазури:

    • 1/5 стакана сахара
    • три чайных ложки какао
    • четыре столовых ложки молока
    • столовая ложка сливочного масла

Приготовление

Делаем крем: сметану взбить с сахаром и глазурь: сахар смешать с какао, добавить молоко, всё варить на малом огне 20-30 минут, в конце варки добавить сливочное масло. Все ингредиенты для теста смешать, раззделить на четыре части , в одну добавить чайную ложку какао и орехи, раскатать и испечь четыре коржа. Смазать кремом, уложить друг на друга, верхний смазать глазурью.
